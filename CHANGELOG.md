# CHANGELOG

## v3 (относительно v2)

Ниже перечислены ключевые изменения, вошедшие в релиз `v3` по сравнению с `v2`.

### Новые возможности

- Добавлен стартовый пресет расположения панелей при первом запуске.
- Улучшено позиционирование панелей: привязка к соседним границам и более надежная привязка к центру.
- Добавлены элементы управления выравниванием панелей в настройках.
- Реализована подготовка и ограничение сохраненных раскладок в рамках экрана (clamp), чтобы панели не "уезжали" за видимую область.
- Bartender4 добавлен в меню интерфейса игры для более быстрого доступа к настройкам.

### Улучшения интерфейса и настроек

- Меню настроек панелей переведено на древовидную структуру для более удобной навигации.
- Переключатель включения панели вынесен в более заметное место.
- Проведена переработка генерации опций статус-баров и скорректированы значения по умолчанию.
- Стабилизирована работа переключения отображения сетки.
- Локализованы подписи для элементов управления выравниванием.

### Исправления

- Исправлены зависания интерфейса в настройках полос опыта и репутации.
- Добавлены отсутствующие ключи локализации `enUS` для параметров полос опыта и репутации.

### Документация

- Добавлена дорожная карта переноса функциональности Bartender2 и обновлен план с учетом выполненных шагов P1.
- Добавлена инструкция по использованию команды `/kb` для настройки биндов.
- Добавлен чеклист настройки позиционирования и FAQ по расстоянию между панелями.
- Добавлена заметка по обходному пути для лагов в настройках XP/Reputation.
